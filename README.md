# TasaDolar

Minimal Venezuelan exchange rate tracker using Astro and build-time data fetch.

Includes BCV USD/EUR and Binance USDT rates with a lightweight converter.

## Commands

| Command             | Action                                       |
| :------------------ | :------------------------------------------- |
| `npm install`       | Install dependencies                         |
| `npm run dev`       | Start dev server at `localhost:4321`         |
| `npm run build`     | Build production site to `./dist/`           |
| `npm run preview`   | Preview build locally                        |
| `npm run astro ...` | Run Astro CLI commands (`add`, `check`, etc) |
